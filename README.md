# nhentai-api

nhentai-api is a basic and easy to use API for [nhentai.net](https://nhentai.net)

## Installation

Gem install
```
gem install nhentai-api
```

Gemfile
```
gem 'nhentai-api'
```

## API
```
doujinshi = Doujinshi.new(220794)
doujinshi.title
doujinshi.pages
...
```
